# Space Odyssey: Galactic Evolution - Game Guide

## Overview

Space Odyssey is a mobile-first idle RPG where you guide a civilization through multiple eras of technological and cultural evolution. Mine elements, build structures, research technologies, and navigate diplomatic encounters with alien factions.

## Core Gameplay Loop

### 1. Resource Generation (Idle)
- **Automatic Mining**: Your mining operations automatically generate resources over time
- **Offline Progress**: When you close the app, your civilization continues generating resources
- **Sync on Login**: All offline progress is automatically calculated and synced when you return

### 2. Element Mining
- **16 Planet Zones**: Each zone contains different elements with varying rarity
- **Mining Types**:
  - **Safe Mining**: Low risk, consistent yields
  - **Aggressive Mining**: Higher yield, moderate risk
  - **Deep Core Mining**: Maximum yield, high risk of failure
- **50+ Elements**: Discover and collect elements from the periodic table
- **Rarity Tiers**: Common → Uncommon → Rare → Epic → Legendary

### 3. Building & Infrastructure
- **8 Building Types**:
  - **Mines**: Increase element extraction rates
  - **Labs**: Enable technology research
  - **Habitats**: Support population growth
  - **Defense**: Protect against invasions
  - **Storage**: Increase resource capacity
  - **Refineries**: Process raw elements
  - **Temples**: Unlock cultural bonuses
  - **Trade Posts**: Enable faction trading

- **Upgrade System**: Each building can be upgraded to increase effectiveness
- **Resource Costs**: Buildings require elements and credits to construct

### 4. Technology Research
- **30+ Technologies**: Unlock new capabilities across 4 eras
- **Tech Tree**: Prerequisites determine research order
- **Research Bonuses**: Completed techs provide permanent bonuses
- **Era Progression**: Advance through Stone Age → Bronze Age → Industrial → Atomic

### 5. Civilization Evolution
- **4 Trait Types**:
  - **Militaristic**: Combat bonuses, unit production
  - **Scientific**: Research speed, element discovery
  - **Stealth**: Espionage bonuses, evasion
  - **Industrial**: Building efficiency, resource production

- **Traits Unlock**: Traits are earned through gameplay choices and achievements

### 6. Narrative Events
- **AI-Generated Stories**: Each event is uniquely generated using LLM
- **Branching Choices**: 2-3 options per event with different consequences
- **Context Awareness**: Events reflect your current civilization state
- **Consequences**: Choices affect resources, relationships, and progression

### 7. Combat & Espionage
- **Fleet Battles**: Turn-based combat with unit composition
- **Unit Types**: Fighters, Bombers, Capitals, Scouts
- **Combat Actions**: Attack, Defend, Retreat
- **Espionage Missions**:
  - **Scan**: Gather intelligence (90% success)
  - **Spy**: Plant agents (60% success)
  - **Disrupt**: Sabotage operations (50% success)
  - **Fake Signals**: Confuse enemies (70% success)

### 8. Faction System
- **Alien Factions**: Multiple civilizations with distinct personalities
- **Reputation System**: -100 to +100 reputation per faction
- **Relationship Tiers**: Hostile → Neutral → Friendly → Allied
- **Diplomatic Options**: Trade, ally, threaten, negotiate

## Progression Systems

### Daily Rewards
- **7-Day Streak**: Claim daily rewards to maintain streaks
- **Escalating Rewards**: Day 1-6 increasing resources, Day 7 prestige token
- **Reset on Miss**: Streak resets if you miss a day

### Achievements
- **10+ Achievements**: Unlock through gameplay milestones
- **Rarity Tiers**: Common → Uncommon → Rare → Epic → Legendary
- **Prestige Points**: Achievements award prestige points

### Prestige System
- **Reset Mechanic**: Voluntarily reset your civilization for permanent bonuses
- **Prestige Multiplier**: 10% resource multiplier per prestige level
- **Progression Resets**: Buildings, techs, and elements reset
- **Permanent Bonuses**: Multipliers persist across resets

## Mobile Interface

### Bottom Navigation (7 Tabs)
1. **Planet**: Interactive mining map with zone selection
2. **Buildings**: Construct and upgrade structures
3. **Tech**: Research technologies in the tech tree
4. **Events**: View and respond to narrative events
5. **Combat**: Engage factions in battle
6. **Achievements**: Track progress and unlocked achievements
7. **Settings**: Manage daily rewards, prestige, and game stats

- **Royal Blue Background**: Deep technical atmosphere
- **Grid Pattern**: Engineering precision overlay

- **Technical Typography**: Bold, sans-serif fonts

### Touch Optimization
- **Large Tap Targets**: All buttons sized for mobile touch
- **Responsive Layout**: Adapts to all screen sizes
- **Swipe Navigation**: Smooth transitions between tabs

## Tips & Strategies

### Early Game (Era 1)
- Focus on mining common elements
- Build mines to increase production
- Research basic techs to unlock new zones
- Complete early achievements for prestige points

### Mid Game (Era 2-3)
- Diversify element collection
- Build multiple building types for synergy
- Establish faction relationships through events
- Engage in light combat/espionage

### Late Game (Era 4+)
- Optimize building placement for maximum efficiency
- Master combat strategy with fleet composition
- Pursue prestige resets for permanent bonuses
- Complete all achievements for maximum prestige

### Prestige Strategy
- Reset when you've reached a natural progression plateau
- Each prestige level provides 10% resource multiplier
- Plan resets to maximize long-term progression
- Prestige becomes essential for late-game advancement

## Game Balance

### Resource Economy
- Elements are the primary currency
- Credits are secondary, earned from mining
- Building costs scale with civilization level
- Research costs increase with tech tier

### Combat Balance
- No single unit type dominates
- Strategy matters more than raw numbers
- Espionage provides alternative to direct combat
- Diplomatic options available for all encounters

### Progression Pacing
- Early game: 5-10 minutes per session
- Mid game: 10-30 minutes per session
- Late game: 30+ minutes or idle progression
- Prestige cycles: 2-4 hours per cycle

## Offline Mechanics

### Offline Progress
- Resources continue generating while offline
- Buildings continue producing at normal rates
- No penalties for being offline
- Progress capped at 24 hours maximum

### Sync System
- Automatic sync on login
- Conflict resolution favors player
- Timestamp tracking for accuracy
- No manual sync required

## Accessibility

- **Colorblind Mode**: Available in settings
- **Text Scaling**: Adjustable font sizes
- **High Contrast**: Blueprint aesthetic provides high contrast
- **Keyboard Navigation**: Full keyboard support
- **Screen Reader**: Compatible with accessibility tools

## Troubleshooting

### Offline Progress Not Syncing
- Check internet connection
- Restart the app
- Clear app cache and reload

### Combat Not Initiating
- Ensure you have discovered factions
- Check faction relationship status
- Verify you have units available

### Events Not Generating
- Events generate periodically
- Check event log for recent events
- Prestige resets event history
