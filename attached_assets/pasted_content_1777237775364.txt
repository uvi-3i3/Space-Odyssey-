🌌 SPACE ODYSSEY — GAME DESIGN DOCUMENT
1. 🎯 Game Overview
Genre:
Idle RPG + Strategy + Text-driven Narrative + Asynchronous Multiplayer
Platform:
Mobile-first (portrait), web compatible
Core Fantasy:
You start as a lone space explorer and evolve into the ruler of a multi-planet civilization, discovering elements, shaping lifeforms, and outsmarting other players through strategy, espionage, and expansion.
2. 🔁 Core Gameplay Loop
This is the heart of the game — everything feeds into this:
Primary Loop:
Explore / Scan
Mine / Collect Elements
Research / Combine Elements
Build / Upgrade
Train Units / Evolve Species
Spy / Invade
Earn Rewards → Repeat
Micro Loop (Moment-to-moment):
Tap planet → scan result
Choose mining method
Trigger event → make decision
Get reward / consequence
👉 The goal: constant small decisions + visible progress

🧬 Phase 3: Evolution

⚔️ Phase 4: Expansion & Conflict
Build fleets
Spy on others
Launch invasions
🌌 Phase 5: Galactic Influence (Endgame)
Control multiple planets
Rare elements
Alliances / politics
